# Casamento - Guilherme & Marcella

Conteúdo pronto para publicar no GitHub Pages.

Arquivos incluídos:
- index.html
- padrinhos.html
- cronograma.html
- rsvp.html
- presentes.html
- styles.css
- script.js
- /img (logo e placeholders)
- apps_script.gs (Google Apps Script para integrar com Google Sheets)

## Como usar
1. Abra `apps_script.gs` no Apps Script (Extensões → Apps Script) e substitua EMAIL_NOTIFY e ADMIN_TOKEN. Faça o deploy como Web App (Executar como: você; Acesso: Qualquer pessoa). Copie a URL e cole em `script.js` como SHEET_URL.
2. Substitua `PIX_KEY` em `script.js` pela sua chave Pix.
3. Substitua as imagens em `/img` por imagens reais (logo-nomes.png, pres1.jpg...)
4. Faça upload dos arquivos para o repositório GitHub (pasta root). Ative GitHub Pages em Settings → Pages → main branch.

Pronto! O site estará disponível em https://<seu-usuario>.github.io/<repo>/
